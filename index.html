Projects

While in university, I have embarked on various engineering projects through modules and on my student exchange programme as well.

Simulation of breathing through mask

- Project for ME2135, to simulate breathing through an Air+ SMART mask
- Using ANSYS Fluent, simulated 2D breathing with and without mask, and final 3D breathing with a mask
- Models were CAD using SolidWorks, simulations run on ANSYS Fluent using turbulence models
- Determined that using Air+ SMART mask allows for better airflow
